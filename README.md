# 全球媒体风险洞察

分析各国主流媒体过去 24 小时对中国构成风险的报道。

## 一键部署到 Render（推荐）

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/CherieYingXue/global-media-risk)

部署完成后手机访问：**https://global-media-risk.onrender.com**

## 本地运行

```bash
cd risk_insight
pip install -r requirements.txt
python run.py
```

详见 [risk_insight/DEPLOY_RENDER.md](risk_insight/DEPLOY_RENDER.md)
